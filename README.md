# Mloun-Side-Contact
A fully functional side-hover slide-out plugin that supports multiple contact methods with customization options.

== Description ==

Mloun Side Contact is a powerful WordPress plugin that adds a customizable floating contact sidebar to your website. It supports multiple contact methods including phone, email, social media, and custom links with QR code display.

**Key Features:**

* **Multiple Contact Methods** - Phone, Email, WeChat, QQ, WhatsApp, Telegram, TikTok, X (Twitter), Facebook, and custom links
* **Custom Icon Upload** - Upload your own icons for personalized branding
* **QR Code Display** - Show QR codes on hover for easy mobile scanning
* **Flexible Positioning** - Choose left or right side, adjust top distance and offset
* **Color Customization** - Customize background colors, hover colors for each item
* **Mobile Responsive** - Automatically converts to bottom bar on mobile devices
* **Drag & Drop Sorting** - Easily reorder contact items via drag and drop
* **Multi-language Support** - Ready for translation with POT file included

**Perfect for:**

* Business websites
* E-commerce stores
* Service providers
* Any website needing quick contact access

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/mloun-side-contact/`, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to 'Side Contact' in the admin menu to configure the plugin.
4. Add your contact items, customize colors and positions.
5. Save settings and view your site to see the floating contact bar.

== Frequently Asked Questions ==

= How do I add a custom icon? =

Select "Custom Icon (Upload)" from the icon dropdown, then upload your image using the upload button. Supported formats: JPG, PNG, SVG.

= Can I change the position of the sidebar? =

Yes, you can choose between left and right position in the settings. You can also adjust the top distance and side offset.

= Does it work on mobile devices? =

Yes, the plugin is fully responsive. On mobile devices, it displays as a fixed bottom bar for better usability.

= Can I hide it on mobile? =

Yes, there is an option to hide the sidebar on mobile devices in the settings.

= How do I add a QR code? =

When editing a contact item, enter the QR code image URL in the "QR Code URL" field. The QR code will display when users hover over the item.

= Can I customize colors for each contact item? =

Yes, each contact item can have its own background color and hover color.

= Is it compatible with page builders? =

Yes, Mloun Side Contact works with all major page builders including Elementor, Divi, Beaver Builder, and WPBakery.

= Does it support RTL languages? =

Yes, the plugin is RTL-ready and will automatically adjust for RTL languages.
